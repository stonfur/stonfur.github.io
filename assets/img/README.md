# Profile photo

Place your own profile photo in this directory and name it exactly:

```text
profile.jpg
```

The homepage references it as `assets/img/profile.jpg`. A portrait-oriented image works best. Do not place another person's photo here.

请将你自己的个人照片命名为 `profile.jpg` 并放在本目录中。首页已经预留头像位置，请勿使用他人的照片。
