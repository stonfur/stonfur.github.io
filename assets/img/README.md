# Profile photo

Place your own profile photo in this directory and name it exactly:

```text
profile.jpg
```

The homepage references it as `assets/img/profile.jpg`. A square or nearly square image works best with the reference-style profile table. Do not place another person's photo here.

请将你自己的个人照片命名为 `profile.jpg` 并放在本目录中。方形或接近方形的照片最适合当前参考页风格的资料表；请勿使用他人的照片。
